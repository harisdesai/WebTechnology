What is WebSocket?
WebSocket is a protocol that allows real-time, two-way (full duplex) communication between:
Client (browser)
Server
Unlike HTTP:
HTTP = request → response (one-time)
WebSocket = continuous connection

Step-by-step:
Client sends HTTP request (handshake)
Server upgrades connection to WebSocket
Connection stays open
Data can flow both ways anytime

ex--->
const socket = new WebSocket("ws://localhost:3000");
socket.onopen = () => {
  console.log("Connected to server");
  socket.send("Hello Server");
};
socket.onmessage = (event) => {
  console.log("Message from server:", event.data);
};
socket.onclose = () => {
  console.log("Connection closed");
};
